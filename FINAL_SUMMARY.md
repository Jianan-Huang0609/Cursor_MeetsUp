## 🚀 如何使用

### 1. 本地开发
```bash
cd cursor-meetup
npm install
npm run dev
```
访问 http://localhost:5173 查看效果

### 2. 构建生产版本
```bash
npm run build
```

### 3. 预览生产版本
```bash
npm run preview
```

## 📊 数据模型

项目使用结构化的 JSON 数据，包含：

- **Event**: 活动基本信息
- **Speakers**: 嘉宾信息和演讲内容
- **Albums**: 照片相册

所有数据都在 `src/data/meetup.json` 中，易于修改和扩展。
